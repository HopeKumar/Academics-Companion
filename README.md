# Academic Companion

## Overview
Academic Companion is an AI-powered LMS enhancement system that personalizes learning, tracks academic progress, and provides intelligent academic insights.

## Tech Stack

### Frontend
- Angular 16.0.2
- Tailwind CSS 3.4.18
- Node.js 18.10.0

### Backend
- Java 17
- Vert.x 4.5.13
- MongoDB 7
- Maven

### UI/UX
- Figma

---

## Project Structure

- frontend/ → Angular application
- backend/ → Vert.x backend services
- ui-ux/ → Design files and prototypes
- docs/ → Documentation
- assets/ → Images and media
