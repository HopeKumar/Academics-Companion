# Frontend

This folder contains the frontend application for **Academic Companion**.

## Tech Stack
- Angular 16.0.2
- Tailwind CSS 3.4.18
- Node.js 18.10.0

## Responsibilities
- UI development using Angular
- Styling with Tailwind CSS
- API integration with backend services
- State and component management

## Structure (recommended)
- src/
- app/
- components/
- pages/
- services/
- assets/

## Setup Instructions

```bash
npm install
ng serve
